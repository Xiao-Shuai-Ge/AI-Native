## 真实模型说明

- 默认测试使用 fake/mock LLM，不依赖付费 API。
- DeepSeek 与 Ollama smoke test 使用 `pytest -m smoke` 隔离。
- OpenAI 与 Claude 通过统一 adapter contract test 验证配置和协议边界，避免课程演示产生不必要费用。

## 已知限制

- Docker Compose 的单服务 `--scale worker=2` 与 Dapr sidecar 一一绑定存在天然限制；本项目默认演示单 worker。若要演示两个 worker，需要显式增加第二组 worker/sidecar 或使用 Kubernetes。
- Web SSE 由 API 进程内广播提供；多 API 副本需要 sticky session，否则详情页会降级轮询。
- `web_search` 依赖外部搜索服务，网络或额度不可用时会影响真实工具结果；自动化测试使用 mock。
